# Financial Optimization & Algorithmic Paradigms
**Course:** Analysis and Design of Algorithms (ADA)
**Project:** Lab Experiment - 1
**Developer:** Anju

## 📌 Problem Context
In the financial sector, decision-making and optimization problems arise across domains such as finance and resource allocation. This project implements various algorithmic paradigms to solve core business problems, such as high-frequency stock price sorting, corporate boardroom scheduling, investment portfolio budgeting, and currency exchange systems.

## 🧠 Algorithmic Strategies
This project explores three core paradigms applied to financial scenarios:

1. Divide and Conquer: Applied via QuickSort to process and sort large volumes of stock market transaction data.
2. Greedy Algorithms: Utilized for Activity Selection to optimize corporate meeting schedules, ensuring the maximum number of non-overlapping tasks are completed.
3. Dynamic Programming: Essential for 0/1 Knapsack (Portfolio Budgeting) and the Change Making problem where overlapping subproblems require optimal substructure for exact solutions.

## 🛠️ Setup & Usage
### Prerequisites
* Python: ≥ 3.10 with venv
* Libraries: matplotlib, numpy, memory_profiler, jupyter, time

### Installation
1. Create a private repository named: algo-strategies-mini-project-Anju
2. Initialize the environment:
   python -m venv .venv
   source .venv/bin/activate  (On Windows: .venv\Scripts\activate)
3. Install dependencies:
   pip install matplotlib numpy memory_profiler jupyterlab

### Usage
Launch the Jupyter Notebook to view the implementations and performance plots:
jupyter lab notebooks/algo_strategies_notebook.ipynb

## 📊 Summary Comparison Table
| Task | Algorithm | Strategy | Complexity | Business Application |
| :--- | :--- | :--- | :--- | :--- |
| Stock Sorting | QuickSort | D&C | O(n log n) | Market Trend Analysis |
| Meeting Scheduling | Activity Selection | Greedy | O(n log n) | Room Resource Management |
| Portfolio Budgeting | 0/1 Knapsack | DP | O(n * Budget) | Investment Maximization |
| Currency Exchange | Change Making | DP | O(n * Amount) | Transaction Efficiency |
